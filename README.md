# Derby: A Coffee E-commerce Website

This is a demonstration project showcasing a coffee e-commerce website built with HTML, CSS, JavaScript, and jQuery. It features 10 unique pages designed to simulate a real online coffee store.


## Live Demo:

- https://zaki-abed.github.io/derby/

## Tech Stack:

- HTML
- CSS
- JavaScript
- jQuery
